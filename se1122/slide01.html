Course Introduction & Syllabus Overview

What We Cover

This course introduces the fundamental concepts of programming with emphasis on the following topics:

  • Variables and Expressions: Assigning variables to expressions and the different types of values.
  • Functions: Units of reusable code that can be called throughout a program.
  • Conditionals and Loops: Ways to change the behavior of the program during runtime.
  • More Advanced Data: String, Arrays, and Dictionaries
