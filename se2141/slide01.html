Database Systems

Course Introduction & Syllabus Overview

What We Cover

This course provides a comprehensive foundation in the design, implementation, and management of data systems:

  • Basics of Databases & DBMSs: Understanding storage, architecture, and the relational model.
  • SQL (Structured Query Language): Writing robust DDL and DML queries to manipulate data.
  • Database Design & Normalization: Organizing tables to prevent redundancy and dependency anomalies.
  • Entity-Relationship Diagrams (ERDs): Mapping conceptual schemas into physical blueprints.

Course Prerequisites

To succeed in SE 2141, you must have successfully completed the following foundation blocks:

  • EMath 1105 (Discrete Math): Relational algebra, set theory, and logic form the direct mathematical underpinnings of relational databases.
  • SE 1242 (Data Structures)

You will not build your database in isolation. This course features an cross-subject framework alignment:

  • Database (SE 2141): You will design the physical schemas, build ERDs, normalize tables, and implement relational constraints.
